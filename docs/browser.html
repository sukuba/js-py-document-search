使えるブラウザ

FireFox および Safari

この2つが推奨ブラウザ。

使えるための要件（その他のブラウザの判定方法）
インターネット標準規格を満たすこと。

いわゆるモダンブラウザで、W3Cなどの規格を順守したものが必要。 キーワードは、 html5, css3, ecma6。 JavaScript で ajax 呼び出しを行うため、スクリプト回りの性能が極端に低いものだと、 速度が出ないかもしれない。 Androidスマホ付属の（FireFoxでない）標準的なブラウザでも稼働しており、 このハードルはそれほど高くない。

JsNgramライブラリのオンラインデモページで、稼働ブラウザかどうかをすぐに確かめることができる。（英語）

ローカルファイルの動的呼び出しを許可していること。

セキュリティポリシーの問題。 Google Chrome や Microsoft IE は、ローカルファイルの ajax 呼び出しを制限しているため、 サーバーレスでは使えない。 また、iOS はローカルファイルへのアクセスを強く制限しているため、 iPhone, iPad でも、サーバーレスでは使えない。 このセキュリティポリシー制限だけが問題になるケースでは、 静的なhttpサーバーを用意し、 file:// プロトコルでなく http:// プロトコルを使うことで利用可能となる。

使えないブラウザ

Microsoft IE（マイクロソフト・インターネット・エクスプローラ）

インターネット標準規格を満たさないため、動作しない。
