list_file_size.py

.. 上の階層へ . リファレンスの見出しへ

ユーティリティ（使用していない）

説明

指定したディレクトリの、サブディレクトリも含めたファイル名とサイズの一覧を取得する。 ファイルサイズとファイルへのフルパスをタブ区切りで出力するので、 出力をファイルに保存すれば、エクセルなどで開くことができる。 たとえば pub/db および pub/txt のファイル一覧を取得して、 エクセルで比較するといった用途に使える。 Python 3 用だが、Python 2 でも動くかもしれない。

使用方法 
    List file sizes of source directory, including subdirectories.
    
    list_file_size.py E:\scratch\txt
    
    1st arg: Source directory to read files (fullpath)
    
    Output to console: size(bytes) and fullpath delimited with tab
