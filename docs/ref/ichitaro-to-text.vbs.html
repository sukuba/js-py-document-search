ichitaro-to-text.vbs

.. 上の階層へ . リファレンスの見出しへ

step4用のVBScriptスクリプト

説明

更新用。 一太郎形式の文書ファイルをutf-8テキストファイルに変換する。 ichitaro-to-text.batから直接呼び出される。 Windows専用のVBScriptファイル。

使用方法 
第1引数 : 変換したい一太郎ファイルのフルパス
第2引数 : 変換先フォルダのフルパス
変換先ファイル名は、元のファイル名に .txt を付加したものになる。

実行時オプション
/normalize : unicode正規化を行う。
/nojxw : 一太郎を起動せずダミーデータを使う（一太郎の無い環境でのデバッグ用）
/dump  : 読み込んだテキストをコンソールに出力する（デバッグ用）
/ansi  : 中間ファイルを cp932 で保存する。（デバッグ用）

一太郎を起動し、
指定ファイルを開き、
全テキストを選択して取得する。

テキストを選択するというプロセスが入っているため、
一太郎を開いた時のウィンドウの状況に依存して、
正しく動かない場合がある。

通常の状態で開くように、
ファイルを保存しておけば問題は回避できる。
たとえば、校閲モードなどの特殊な状態を切っておくといった
配慮が必要。
