.. 上の階層へ . リファレンスの見出しへ

インデックスと共に生成される

説明

検索対象文書群の先頭階層にある、収録ファイル一覧*.xlsファイルから、 step3xls.batによって生成される。 検索の結果表示に、タイトルや著者名を付与する際に利用する。 検索結果に一致するキーが、このファイルに無い場合、タイトル等の表示は行われず、文書へのパスがそのまま表示される。

仕様
  • json形式。
  • 文書のパスをキーとしたハッシュ。 これは元のエクセルファイルの、E列のハイパーリンクから取得する。
  • 対応する値は、以下の項目の配列。 タイトル、著者、年度、資料の種類、分別番号など、ユニークID。 これは元のエクセルファイルの、E,F,B,C,D,A列に相当する。
  • 実施課題一覧シートだけを使う。
